##  Project Description:
This project presents an **AI-driven mental health screening system** designed for adolescents and individuals with disabilities. It uses a **multimodal approach**, combining:

-  Speech Analysis  
-  Text Sentiment Analysis  
-  Clinical Questionnaire  

The system integrates **CNN, LSTM, and Random Forest models** to deliver accurate and reliable mental health predictions.

---

##  Objectives:
- Early detection of mental health issues  
- Identify conditions like depression, anxiety, trauma, and phobias  
- Provide personalized coping strategies  
- Enable continuous mental health tracking  

---

##  Functioning and User Flow:

1. **User Input**
   - Enter Name and Email  

2. **Voice Speech Analysis**
   - Tap to speak  
   - System analyzes tone (Normal / Stressed / Anxious)  

3. **Journal Entry (Text Analysis)**
   - User writes feelings  
   - LSTM analyzes sentiment  

4. **Clinical Questionnaire**
   - 11 structured questions  
   - Responses are scored  

5. **Model Processing**
   - Combines all inputs  
   - Hybrid Model: CNN + LSTM + Random Forest  

6. **Result Generation**
   - Displays condition (e.g., Mild Depression)  
   - Shows confidence score  

7. **Recommendations**
   - Breathing exercises  
   - Anxiety relief techniques  
   - Panic control methods  

8. **Progress Tracking**
   - Mood tracking  
   - Exercise tracking  

9. **AI Chatbot**
   - Provides mental health guidance  

---
